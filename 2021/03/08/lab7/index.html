LAB 7 -- วันที่ 8 มีนาคม พ.ศ. 2564

Web Authentication Vulnerabilities

ในแลบนี้นักศึกษาจะใช้ web security dojo VM และทำแบบฝึกหัด WebGoat เรื่อง Authentication Flaws โดยให้ข้ามหัวข้อแรกคือเรื่อง Password Strength

ขั้นตอนการทำแลบ

  1. Boot เครื่องเสมือนชื่อ dojo
  2. ใช้ burp suite และ firefox
  3. ตั้งค่าให้ burp ดักจับ connection ของ firefox
  4. Start WebGoat
  5. ทำแลบ WebGoat ในส่วน Authentication Flaws โดยให้ใช้
    1. Burp (ใช้สำหรับดักจับข้อมูลระหว่าง browser และ server) และ
    2. firefox dev tools (ใช้ในการสำรวจ source code)
    เท่านั้น ห้ามใช้เครื่องมืออำนวยความสะดวกอื่น ๆ เช่น Tamper Data, เว็บไซต์ที่ช่วยในการ encode/decode base64, Live HTTP headers ฯลฯ
  6. เขียนข้อสรุปสั้น ๆ บนกระดาษ A4 สำหรับแบบฝึกหัดแต่ละข้อให้ครบทั้ง 3 ข้อ (Forgot Password, Multi Level Login 1, Multi Level Login 2) โดยในแต่ละข้อให้อธิบายคำตอบในสองประเด็นต่อไปนี้
    1. สาเหตุที่ทำให้เกิดช่องโหว่คืออะไร
    2. วิธีการปิดช่องโหว่หรือรับมือกับช่องโหว่นั้นมีอะไรบ้าง

    7. ข้อมูลที่อาจเป็นประโยชน์

    • นักศึกษาสามารถศึกษาการใช้งานคำสั่ง openssl ได้โดยการพิมพ์ man openssl ที่ command prompt

    สิ่งที่ต้องส่ง

    • คำตอบในข้อสุดท้าย กำหนดส่ง 23.59 น. 9 มีนาคม 2564
