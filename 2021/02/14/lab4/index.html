LAB 4 -- วันที่ 15 กุมภาพันธ์ พ.ศ. 2564

เรียนรู้พื้นฐานที่เกี่ยวข้องกับ HTTP

ให้นักศึกษาปฏิบัติตามวิดีโอ HTTP Crash Course & Exploration แล้วอัดวิดีโออธิบายสิ่งที่นักศึกษาทำ ตามลำดับที่ปรากฎในวิดีโอต้นแบบ

นักศึกษาสามารถคัดลอก code ต่อไปนี้ไปใช้เป็นต้นแบบได้

สิ่งที่นักศึกษาควรเรียนรู้ในแลบนี้

  • ความแตกต่างระหว่าง GET, POST, PUT, DELETE methods
  • ความหมายของ web API และการใช้งาน Postman Client ในการทดสอบ web API
  • การใช้งาน web browser developer tools เช่น Firefox หรือ Chrome
  • การติดตั้งและตั้งค่า node.js และ express.js สำหรับทำการทดลองส่ง request และรับ response และตรวจสอบค่า HTTP messages ที่ส่งไปให้และได้รับจากเซิร์ฟเวอร์

สิ่งที่ต้องศึกษาสำหรับแลบนี้

  • What are APIs? - Simply Explained (ความยาว 6:48 นาที)
  • Postman Client download
  • Postman 101 for Developers (สำหรับวิชานี้ ให้นักศึกษาดูวิดีโอถึงนาทีที่ 16:00 เท่านั้น)
  • How to learn HTTP using Firefox Browser Dev Tools (ความยาว 4:00 นาที)
