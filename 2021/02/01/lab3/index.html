LAB 3 -- วันที่ 8 กุมภาพันธ์ พ.ศ. 2564

ส่วนที่ 1: ศึกษาความแตกต่างระหว่างการแลกเปลี่ยนข้อมูลเมื่อใช้ method GET และ method POST

  1. สร้างไฟล์ form-w-get.html เพื่อแสดงแบบฟอร์มที่มีช่องรับอินพุตเป็น text เพียงหนึ่งช่อง ให้อินพุตนี้มีชื่อว่า form-input และให้ใช้ method เป็น GET
  2. สร้างไฟล์ form-w-post.html เพื่อแสดงแบบฟอร์มที่มีช่องรับอินพุตเป็น text เพียงหนึ่งช่อง ให้อินพุตนี้มีชื่อว่า form-input และให้ใช้ method เป็น POST
  3. สร้างไฟล์ get-formhandler.php และ post-formhandler.php สำหรับประมวลผลข้อมูลที่ได้รับจากการกรอกแบบฟอร์มแบบ GET และแบบ POST ตามลำดับ โดยให้แสดงผลลัพธ์เป็น form-input = x โดย x คือ ค่าที่ได้รับจากการกรอกแบบฟอร์มของผู้ใช้
  4. นำไฟล์ form-w-get.html form-w-post.html get-formhandler.php และไฟล์ post-formhandler.php ไปวางไว้ในที่ที่เหมาะสม เพื่อให้สามารถเข้าถึงได้โดยผ่าน URL http://targets.local/lab2/….
  5. ทดสอบโดยใช้ burp suite และ developer tools ใน firefox
  6. พิจารณาว่าการใช้ GET และ POST นั้นมีความแตกต่างกันอย่างไร แล้วเขียนบรรยายตามความเข้าใจของตนลงในบันทึกแลบ

ส่วนที่ 2: ทดสอบการรับส่ง hidden variables

  1. ใน directory ชื่อ lab2 สร้างไฟล์ hidden-var-form.html เพื่อแสดงแบบฟอร์มที่มีช่องรับอินพุตเป็น text เพียงหนึ่งช่อง ให้อินพุตนี้มีชื่อว่า normal-var และมีการส่ง hidden variable หนึ่งตัวชื่อ hidden-var
  2. สร้างไฟล์ hidden-var-formhandler.php เพื่อประมวลผลข้อมูลที่ได้รับจากการกรอกแบบฟอร์ม โดยให้แสดงผลลัพธ์เป็น normal-var = x and hidden-var = y โดย x คือ ค่าที่ได้รับจากการกรอกแบบฟอร์มของผู้ใช้ และ y คือ ค่าของ hidden variable
  3. นำไฟล์ hidden-var-form.html และไฟล์ hidden-var-formhandler.php ไปวางไว้ในที่ที่เหมาะสม เพื่อให้สามารถเข้าถึงได้โดยผ่าน URL http://targets.local/lab2/….
  4. ทดสอบโดยใช้ burp suite และ developer tools ใน firefox
  5. อธิบายว่า hidden variables คืออะไร การรับ-ส่ง hidden variable มีลักษณะอย่างไร

ส่วนที่ 3: ทดสอบการรับส่ง cookies

  1. สร้างไฟล์ชื่อ setcookie.php โดยให้มีการทำงานคือ ให้สั่ง server ให้ส่ง cookie ที่มี name เป็น remember และ value เป็น ชื่อของนักศึกษา
  2. สร้างไฟล์ชื่อ usecookie.php โดยให้มีการทำงานคือ ให้สั่ง server ให้แสดงค่า cookie ที่ได้รับมาจาก browser
  3. ใช้ burp suite และ firefox quantum ในการสำรวจเนื้อหาของข้อมูลที่มีการแลกเปลี่ยนกันสำหรับทั้งสอง requests ข้างต้น
  4. อธิบายว่า cookies คืออะไร การรับ-ส่ง cookies มีลักษณะอย่างไร
